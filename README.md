# NBA 선수 이상형 월드컵

NBA 선수 사진과 이름을 보고 1:1로 선택하면서 최종 우승 선수를 뽑는 웹 프로젝트입니다.

## 프로젝트 소개
- 32명의 NBA 선수가 랜덤으로 섞여 시작됩니다.
- 매 라운드마다 2명 중 1명을 선택합니다.
- 32강 -> 16강 -> 8강 -> 4강 -> 결승 순서로 진행됩니다.
- 마지막 1명이 최종 우승으로 표시됩니다.

## 주요 기능
- 선수 이름 + 사진 카드 표시
- 32강 토너먼트 진행
- 라운드 자동 진행
- 우승자 결과 화면 출력
- 다시 하기 버튼으로 즉시 재시작
- 모바일/PC 반응형 레이아웃

## 파일 구조
- `index.html`: 화면 구조(헤더, 대결 카드, 우승 화면)
- `style.css`: 전체 디자인 및 반응형 스타일
- `app.js`: 선수 데이터, 셔플, 대결/승자 로직

## 실행 방법
1. 프로젝트 폴더를 엽니다.
2. `index.html` 파일을 브라우저로 실행합니다.
3. 바로 이상형 월드컵을 플레이할 수 있습니다.

## 선수 목록 수정 방법
선수 데이터는 `app.js`의 `players` 배열에서 수정할 수 있습니다.

예시 형식:

```js
{ name: "LeBron James", image: "이미지URL" }
```

- `name`: 화면에 보일 선수 이름
- `image`: 선수 사진 URL

## 참고 사항
- 현재 선수 사진은 `cdn.nba.com` 이미지 URL을 사용합니다.
- 인터넷 연결 상태에 따라 이미지 로딩 속도가 달라질 수 있습니다.
